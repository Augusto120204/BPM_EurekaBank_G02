# Eurekabank - Sistema Bancario

Sistema bancario diseñado para gestionar operaciones financieras básicas.

## Funcionalidades Principales

- **Consultas**: Consultar información de cuentas bancarias
- **Depósitos**: Realizar depósitos en cuentas
- **Retiros**: Realizar retiros de fondos
- **Transferencias**: Transferencias entre cuentas
